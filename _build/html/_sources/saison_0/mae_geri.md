# Mae Geri Kekomi

|Niveau | Age | Objectif annuel | Cycle | Séance |
|:-:|:-:|:-:|:-:|:-:|
|débutant | 5-7 ans | ceinture blanc-jaune | II/V | 3/5 |

## Objectifs principaux
- MAE GERI KEKOMI
- ajuster la distance
- travailler en ligne

 
## Matériel
- N cerceaux
- N cibles
- 2 gants
