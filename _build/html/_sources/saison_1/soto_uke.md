# Soto Uke

|Niveau | Age | Objectif annuel | Cycle | Séance |

## Objectifs principaux :
- Découvrir le geste de SOTO UKE
- Poursuivre la découverte du travail GO NO SEN (riposte GYAKU)

## Matériel
